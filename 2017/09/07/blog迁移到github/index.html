blog迁移到github

Sep 7, 2017

之前是想直接把原有vue搭建的页面直接丢到github上
但是由于我没有搞https，所以在github上打开博客会发生报错。说我在https页面，却去我的服务器上 请求http接口。
然后就有两个选择
1.申请域名，搞htpps
2.

1.https://hexo.io/zh-cn/docs/deployment.html#Git

2.安装 Hexo
所有必备的应用程序安装完成后，即可使用 npm 安装 Hexo。
$ npm install -g hexo-cli

3.新建工程
$ cd blog
$ hexo init .
$ npm install
$ hexo server
便可以在 http://localhost:4000/ 预览

4.更换主题
在https://hexo.io/themes/ 找一个喜欢的主题。
比如apollo https://github.com/pinggod/hexo-theme-apollo 按照提示操作
1.安装主题插件，下载主题文件
$ npm install –save hexo-renderer-jade hexo-generator-feed hexo-generator-sitemap hexo-browsersync hexo-generator-archive
$ git clone https://github.com/pinggod/hexo-theme-apollo.git themes/apollo

2./修改 _config.yml 的 theme 配置项为 apollo:
theme: apollo

# 在归档页面显示所有文章
# 需要上面安装的 hexo-generator-archive 插件支持
archive_generator:
    per_page: 0
    yearly: false
    monthly: false
    daily: false

3.$ hexo server 可以在 http://localhost:4000/ 预览新的主题

5.部署到github
1.安装 hexo-deployer-git。
$ npm install hexo-deployer-git –save

2./修改 _config.yml 文件
deploy:
 type: git
 repository: https://github.com/ChenZhangQuan/ChenZhangQuan.github.io.git
 branch: master
3.打开 https://chenzhangquan.github.io/ 浏览blog

6.平时写博客时候的操作
1.在 /source/_posts 目录下生成md文件，编辑文章。
2.hexo s实时查看文章
3.hexo g生成静态文件
4.hexo d推到github
5.github预览

7.附录 markdown语法使用链接https://www.appinn.com/markdown/
